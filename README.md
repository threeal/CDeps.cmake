# CDeps.cmake

CDeps.cmake is a [CMake](https://cmake.org/) module that provides utility functions for building and installing missing dependencies in your CMake project

## Key Features

* Download external projects using [Git](https://git-scm.com/).
* Build and install external projects using CMake.
* Simple syntax and easy integration.
